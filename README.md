# ALBOOM App 

ALBOOM app uses Google Driveâ„¢ API to create folders for photo albums. ALBOOM allows you to create a Drive folder for an upcoming or past event. You can share this Drive folder with friends allowing them to add media content they have gathered.


# Instructions:
Log in with your google account.
Create an ALBOOM (google drive folder) 
Share that ALBOOM with friends and family
Add some content from gallery or the camera via ALBOOM app!
